# MINISHELL  

Minishell is a bash based terminal


## Requirements  

* Install Readline library
* Clang
```sh
sudo apt-get update && sudo apt-get upgrade
sudo apt-get install libreadline-dev
sudo apt-get install clang 
```  

## How to use

```sh
make
```

```
./minishell
```
